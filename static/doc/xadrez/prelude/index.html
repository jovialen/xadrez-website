Module xadrez::prelude

source · [−]
Expand description

The Xadrez prelude module provides a convenient way to bring common Xadrez types and traits into scope. It re-exports the most commonly used items from the library modules.

Usage:

use xadrez::prelude::*;

This will bring the most commonly used types and traits into scope. Note that not all types and traits are exported by this module, so some items may need to be brought into scope manually.

For more information on the available types and traits, see the corresponding modules.
